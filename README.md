# Official Baccarat Hub

本仓库用于归档和发布多个独立的 HTML 页面。  
这些页面以静态文件的形式存放，不针对任何特定域名或具体网站，仅供信息存档与参考。

## 项目简介

Official Baccarat Hub 是一个以 HTML 页面为主要内容的归档仓库。  
所有页面均为独立文件，不依赖外部数据库或后端服务，可直接在浏览器中打开查看。

本仓库不提供、不推广、不指向任何具体服务或产品，仅作为历史页面资料的存储与展示空间。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- **pages/**：每个 HTML 文件为一个独立页面，文件名反映其内容主题。
- **assets/**：与页面配套使用的资源文件，按类型或用途组织。
